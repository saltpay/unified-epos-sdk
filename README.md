# Teya Unified ePOS SDK Samples

Sample apps demonstrating integration with the [Teya Unified ePOS SDK](https://docs.teya.com/epos-sdk/overview). Each app showcases a simple point-of-sale flow, browsing a product catalog, adding items to a basket, processing a payment with optional tip, and printing a receipt.

## Sample Apps

| App                                                           | Platform | SDK Type | Tech Stack | Key Features |
|---------------------------------------------------------------|----------|----------|------------|--------------|
| [Android AIO](android-samples/aio/EposAioExample)             | Android | All-In-One | Kotlin, Jetpack Compose | Payment, receipt printing |
| [Android PosLink](android-samples/poslink/EposPosLinkExample) | Android | PosLink | Kotlin, Jetpack Compose | Payment, receipt printing, auth management |
| [iOS PosLink](ios-samples/poslink/EposPosLinkExample)         | iOS | PosLink | Swift, SwiftUI | Payment, receipt printing, auth management |

## Getting Started

Pick a sample app for your platform and SDK type, then follow the README in the corresponding directory:

- [Android samples](android-samples/)
- [iOS samples](ios-samples/)
