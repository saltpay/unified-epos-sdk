# Android Samples

Sample Android apps demonstrating integration with the [Teya Unified ePOS SDK](https://docs.teya.com/epos-sdk/overview).

| App                                   | SDK Type | Description |
|---------------------------------------|----------|-------------|
| [All-In-One](aio/EposAioExample)      | AIO | For apps running directly on a Teya All-In-One terminal |
| [PosLink](poslink/EposPosLinkExample) | PosLink | For apps running on a separate device linked to a Teya terminal |
